## 🚀 Try DDEV Today

Ready to simplify your local development? 

```bash
# Install DDEV
curl -fsSL https://ddev.com/install.sh | bash

# Start your first project
mkdir my-project && cd my-project
ddev config --project-type=php
ddev start
```
